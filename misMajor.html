Management Inforamtion System Major

(Digital Business Management)

The Management Information System (MIS) major, as known as Digital Business Management (DBM) major, which the name was changed as the cirriculum has been redesigned.

One of the main reasons that make me love and choose this major is that I get to learn coding, play around with techonologies, and get to study with open-minded and helpful instructors, making my university experience become wonderful. :)

Study Tracks in Management Information System Major

This section will dicuss the main plan (Plan A) of MIS which is the one that I'm interested in the most, there are 2 main tracks in this MIS main plan, Software Design and Development and Data Science.

Track A - Software Design and Development

Software Design and Development track is my most favorite track in MIS major! If you love coding, go for it! This would be the true answer for anyone who is interested in building mobile applications or web applications.

Followings are some courses from Track A that I think they are very interesting to show you!

Track A - Software Design and Development - Course List

MIS4132 Mobile Applications

In the class, we learn building mobile applications by using a framework called Flutter, using Dart programming language.

MIS4133 Software Engineering

In this software engineering course, we study about agile software development, quirements analysis, software design, and coding practices.

MIS4134 UI/UX Design

In UI/UX Design, we learn how to make the websites and applications become visually appealing as well as ehnance user experience.

MIS4231 Web Applications Development

In Web Applications Development, we learn how to develop web applications using ASP.NET MVC, which is a framework using C#.


Track B - Data Science

Data Science track is also a very interesting track to go with. As we know that, nowadays everything is driven by data, and data professionals are now very on demand and high salaries to offer. With this track, we will able to perform different kinds of data analysis, making data visualizations as well as learning how to manage data.

Followings are some courses from Track B that I think they are very interesting to go with.

Track B - Data Science - Course List

MIS4221 Data Science

We learn python as a tool to store, clean, manipulate, visualize, model, and extract information from massive amounts of data from the business perspective.

MIS4124 Data Mining

In this Data Mining course, we use Orange to do some data mining processes, such as preprocessing, forming up models, training models, and so on.

MIS4125 Data Visualization

We learn data visualization for business professionals and techniques in information visualization, and how to best leverage visualization methods.

MIS4142 Big Data

We learn data tools, organization, storage, retrieval, analysis and knowledge discovery at scale. Cloud computing, data storage systems, large-scale data analysis, self-descriptive data representations, and semi-structured data models
